# Pankaj Kumar Sao - Personal Website

Welcome to my personal website! This site showcases my skills, experience, and projects. It's built to serve as an online portfolio and a hub for my professional presence.

## Features
- Portfolio of skills and experience
- Downloadable resume
- Project highlights
- Social media integration
- Responsive design

## Technologies Used
- HTML, CSS, JavaScript
- FontAwesome for icons
- SASS for advanced styling

## Getting Started
To run the website locally:
1. Clone this repository:
   ```powershell
   git clone https://github.com/pankajkumarsao/pankajkumarsao.github.io.git
   ```
2. Open the folder in your browser (no build step required).

## Folder Structure
- `assets/` - CSS, JS, fonts, and SASS files
- `images/` and `img/` - Image assets
- `resume/` - Resume PDF
- `extent/` - Additional resources
